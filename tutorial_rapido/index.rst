============================
Tutorial Rápido para Pyramid
============================

Pyramid es un framework web para Python 2 y 3. Este da un recorrido de
alto nivel y compatible con Python 2 y 3 por las características más
resaltantes.

Este tutorial práctico cubre "un poco de mucho": una introducción a las
herramientas de uso más comun. Divertido, de ritmo veloz, y ciertamente
no dirigido a expertos en el framework web Pyramid.
